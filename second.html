Question-1: What are the different ways to select an element in the DOM?

You can select elements using methods like getElementById, getElementsByClassName, getElementsByTagName, and querySelector/querySelectorAll. These methods allow you to target elements based on their ID, class, tag name, or CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML content within an element, including HTML tags. innerText gets or sets the visible text content, respecting rendered styles. textContent gets or sets all text content, including hidden text, without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation involves attaching a single event listener to a parent element to handle events for all its descendant elements. This approach improves performance by reducing the number of event listeners needed.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates up to its parent elements in the DOM tree. This allows parent elements to handle events that occur on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

You create elements with document.createElement(), add them with appendChild(), and remove them with removeChild(). These methods enable dynamic manipulation of the DOM structure.
